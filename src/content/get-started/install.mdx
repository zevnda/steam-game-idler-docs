# Install

## Steps

<References page='/get-started/install'>
    <Steps>
        ### Download
        Download the `.msi` installer from the [latest release](https://github.com/zevnda/steam-game-idler/releases/latest/) page on GitHub

        <Screenshot>
            <Image src="/examples/install-1.png" />
        </Screenshot>

        ### Install
        Run the `.msi` installer and follow the steps of the installation guide

        ### Run
        After installation, you can launch SGI in one of three ways:
            * Select "Launch Steam Game Idler" and click "Finish" when prompted
            * Run `Steam Game Idler.exe` from the installation directory. Defaults to `C:/Program Files/Steam Game Idler`
            * Use the created desktop shortcut
    </Steps>
</References>

# Auto Idler

<References>
    Games added to the `auto idle list` will be automatically idled when SGI is launched.

    This makes for a hassle-free playtime boosting experience when paired with the `run at startup` option in [Settings > General](/settings/general).

    <Callout type="info">
        You can add upto 32 games games to the auto idle list. Steam imposes this
        limit for simultaneously running games
    </Callout>

    ## Steps

    <Steps>
        {<h3 />}
        Click <Icon type='game' /> in the `sidebar`

        {<h3 />}
        Hover over the `game card` of the game you wish to unlock achievements for and click <Icon type='dots' />

        {<h3 />}
        Click on the <MockButton type='context-add' content='add to auto idle' /> option to add the game to your `auto idle list`

        {<h3 />}
        These games will be automatically idled the next time SGI launches
    </Steps>
</References>

# Auto Game Idler

Games added to the Auto Idle list will be automatically idled when Steam Game Idler is launched.

This makes for a hassle-free playtime boosting experience when paired with the `run at startup` option in [Settings > General](/settings/general).

<Callout type="info">
    A maximum of 32 games can be added to the `auto idle` list, as this is the limit Steam imposes on simultaneously running games.
</Callout>

<Steps>
{<h3>Step 1</h3>}
Click <Icon type='game' /> in the `sidebar`

{<h3>Step 2</h3>}
Hover over the `game card` of the game you wish to unlock achievements for and click <Icon type='dots' />

{<h3>Step 3</h3>}
Click on the `add to auto idle` option to add the game to your `auto idle list`
 
{<h3>Step 4</h3>}
These games will be automatically idled the next time SGI launches
</Steps>

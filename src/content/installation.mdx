# Installation

<Steps>
{<h3>Download</h3>}
Download the `.msi` installer from the [latest release](https://github.com/probablyraging/steam-game-idler/releases/latest/) page on GitHub

{<h3>Install</h3>}
Run the `.msi` installer and follow the steps of the installation guide

{<h3>Run</h3>}
After installation, you can launch Steam Game Idler (SGI) in one of three ways:
1. Select "Launch Steam Steam Game Idler" and click "Finish" when prompted
2. Run `Steam Game Idler.exe` from the installation directory. Defaults to `C:/Program Files/Steam Game Idler`
3. Use the created desktop shortcut
</Steps>

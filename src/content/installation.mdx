# Installation

<Steps>
{<h3>Download</h3>}
Download the `.msi` installer from the [latest release](https://github.com/probablyraging/steam-game-idler/releases/latest/) page on GitHub

{<h3>Install</h3>}
Run the `.msi` installer and follow the steps of the installation guide

{<h3>Run</h3>}
After the installation process has finished you will be prompted to run Steam Game Idler, check the box and click finish

Or, the installer should create a shortcut on your desktop that you can run

Or, you can navigate to the installation path you chose in the installer. By default it is `C:/Program Files/Steam Game Idler`
</Steps>
